
# CRUD com NestJS

CRUD com NestJS.


## Dependência

[Docker](https://www.docker.com/products/docker-desktop/)


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/brunojcamargo/crud-nestjs.git
```

Entre no diretório do projeto

```bash
  cd crud-nestjs
```

Inicie o servidor

```bash
  docker compose up
```


## Documentação API

Após rodar o projeto, acesse http://localhost:3000/api
